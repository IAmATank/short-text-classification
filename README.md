# short-text-classification
SVM, FastText, TextCNN, BiGRU, CNN-BiGRU在短分本分类上的对比

本次对比是采用20w的数据进行分析对比，项目中并没有给出该数据，corpus下的语料只是toy corpus. just for demo.

## environmnet 环境
1. Centos
2. Python3.6
3. PyTorch (latest version)
